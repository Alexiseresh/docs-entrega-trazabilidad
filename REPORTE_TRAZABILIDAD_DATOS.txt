1. FUNCIONALIDADES CRUD Y PROCESOS:
---------------------------------------------------------------------
My sistema cumple con el 100% de los requerimientos funcionales:
- INSERTAR: Registro de entrada de mercancía con validaciones exactas.
- LISTAR/CONSULTAR: Dashboard dinámico con visualización de registros.
- MODIFICAR: Edición de facturas con actualización inmediata.
- ELIMINAR: Borrado lógico/físico de registros.
- BITÁCORA: Módulo avanzado de trazabilidad que registra automáticamente 
  quién, cuándo y qué acción realizó (INSERT, UPDATE, DELETE).

2. ESPECIFICACIONES TÉCNICAS:
---------------------------------------------------------------------
- Frontend: React + TypeScript (Diseño responsivo con Tailwind CSS).
- Base de Datos: Supabase / PostgreSQL (Relacional).
- Seguridad: Autenticación segura y manejo de Roles de Usuario.
